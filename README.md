# Account_Service

- Spring WEB MVC를 이용해서 계좌 시스템의 기본적인 비즈니스 로직과 API 기능을 개발한 토이프로젝트
- Spring MVC의 Model-View-Controller의 기본적인 아키텍쳐를 구현
- TDD 방식을 적용하여 개발을 진행

## 활용 기술 스택
- Spring WEB MVC
- Junit5
- JPA
- H2 DataBase
- Redis
- Lombok
- Mockito
- Intellij IDE
